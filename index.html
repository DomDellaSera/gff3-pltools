gff3-pltools

Fast parallel GFF3 tools

Download trunk as .zip Download trunk as .tar.gz View on GitHub

This is currently an early work in progress to create parallel GFF3 and GTF tools in D and a Ruby gem which would let Ruby programmers use those tools from Ruby.

See README for more information.
